# APD GitHub Training Demo Repository

This repository is a safe, non-production training repository for APD CAD and Structural employees learning the GitHub Desktop workflow.

It contains small example files only:

- `CAD/AutoLISP/` - sample AutoLISP scripts for AutoCAD
- `Structural/VBScript/` - sample VBScript automation examples
- `Structural/VBA/Modules/` - exported VBA source modules such as `.bas` and `.cls`
- `docs/` - workflow notes and training references

## Training purpose

Use this repository to practice:

1. Pulling the latest version.
2. Creating a branch.
3. Editing a script.
4. Testing locally.
5. Reviewing changed files.
6. Committing with a clear message.
7. Pushing the branch.
8. Opening a pull request.
9. Reviewing and merging approved changes.
10. Pulling the completed change back to your computer.

## Security reminder

Do not store passwords, API keys, access tokens, private keys, client data, confidential project information, AutoCAD drawings, or large project datasets in this repository.

Use placeholder values in sample scripts. Store real secrets only in approved APD secure systems.
